Usage Examples

Generate 10,000 industrial control system syslog entries (default):
uv run https://aronchick.github.io/sample-data/generate.py
Generate Windows Event Logs:
uv run https://aronchick.github.io/sample-data/generate.py \ windows
Generate 50,000 cloud manager logs to custom file:
uv run https://aronchick.github.io/sample-data/generate.py \ cloud-mgr -o cloud.log -n 50000
Generate hypervisor logs:
uv run https://aronchick.github.io/sample-data/generate.py \ hypervisor -o hypervisor.log
Stream to stdout (for piping to other commands):
uv run https://aronchick.github.io/sample-data/generate.py \ industrial -o - -n 100 | grep "ERROR"
Show help and all options:
uv run https://aronchick.github.io/sample-data/generate.py \ --help
