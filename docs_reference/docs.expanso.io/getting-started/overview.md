# Expanso Getting Started Documentation

> Source: https://docs.expanso.io/getting-started/

## Overview

Expanso is a managed platform for deploying intelligent data pipelines at the edge. The Getting Started section provides structured guidance for new users to quickly adopt the platform.

## Key Sections

### Quick Start
"Build your first data pipeline in 5 minutes" by learning how to create a network, deploy a node, build a pipeline visually, and monitor data flow. Estimated duration: 5-10 minutes.

### Core Concepts
This module covers foundational knowledge including Expanso's architecture, networks, nodes, pipelines, and edge-first processing benefits. Estimated duration: 15-20 minutes.

### Installation
Instructions for deploying nodes across infrastructure, covering installation methods for all platforms, configuration options, service deployment, and fleet patterns. Estimated duration: 10-30 minutes.

## Recommended Learning Path

1. Quick Start (hands-on introduction)
2. Core Concepts (foundational understanding)
3. Installation (deployment to your infrastructure)
4. Components exploration (200+ available components)
5. Use Cases (real-world applications)

## Key Features Highlighted

- 200+ pre-built components (inputs, processors, outputs)
- Support for AI/ML models (ONNX, TensorFlow Lite)
- Visual drag-and-drop pipeline builder
- Data governance: PII detection, masking, RBAC, SSO
- Comprehensive audit trails
